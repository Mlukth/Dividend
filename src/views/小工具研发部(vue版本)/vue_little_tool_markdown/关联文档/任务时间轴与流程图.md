# 任务时间轴与流程图 (TaskCenterPro)

## 功能概述
集成**任务管理中心**、**时间轴视图**、**泳道热度图**、**任务内嵌决策树流程图**、**邮件任务同步**的综合任务管理工具。

## 核心功能

### 1. 任务管理
- 需求描述输入 + 一键生成提示词复制
- 智能任务导入（粘贴 AI 返回的 JSON 数组）
- 任务卡片：名称、日期、优先级、耗时、目标、标签、截止提醒
- 一键删除已完成任务、撤回导入

### 2. 视图切换
- **时间轴视图**：按日期分组 + **按相似度智能分组**（中文 bigram Jaccard）
- **泳道热度图**：任务耗时条形图 + 截止日期热度着色（过期红/3天橙/7天绿/更远灰）

### 3. 决策树
- 全屏 Vue Flow 决策树流程图，分层布局算法
- 节点增删改、勾选完成状态、分支条件标注
- AI 提示词生成：自然语言描述 → 决策树 JSON
- 进度条实时显示

### 4. 数据持久化（双向同步）
- **localStorage**：浏览器本地存储（保留兼容）
- **task-sync.json**：Vite 中间件 `/api/task-sync` GET/POST 双向同步
- Claude 读/写 task-sync.json → 前端启动加载 → 用户操作写回文件
- JSON 文件导入/导出（项目级含提示词配置）
- 提示词配置持久化

### 5. Claude 集成
- 启动时从 `/api/task-sync` 自动加载 Claude 生成的任务卡片
- 决策树进度合并：同名任务按节点文本匹配更新完成态
- `[Claude-Task]` 邮件任务可通过 checkmail.mjs --tasks 扫描入站

### 6. 辅助功能
- 担忧事项面板（折叠式）
- 截止日期浏览器通知提醒
- 目标语料库（可在编辑时自动补全）

## 技术栈
- Vue 3 + `<script setup>`
- Element Plus UI
- Vue Flow（决策树流程图）
- Vite 自定义中间件（文件读写 API）
- localStorage + JSON 文件双重持久化

## 文件位置
- 组件：`src/views/小工具研发部(vue版本)/TaskCenterPro.vue`
- 数据：`public/task-sync.json`
- API：`/api/task-sync`（Vite 中间件，定义在 vite.config.js）

## 外部脚本
- `D:/scripts/sendmail.py` — 发确认邮件
- `D:/scripts/checkmail.mjs` — 收回复 + 扫描 [Claude-Task] 邮件
- `D:/scripts/README.md` — 邮件脚本说明
