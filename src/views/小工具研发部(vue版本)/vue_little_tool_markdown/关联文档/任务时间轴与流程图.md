# 任务时间轴与流程图

## 功能概述
一个集成**任务管理中心**、**时间轴视图**、**任务流程图**以及**任务内嵌决策树流程图**的综合任务管理工具。

## 核心功能

### 1. 任务管理
- 支持手动输入或 AI 生成的任务 JSON 导入
- 任务卡片展示基本信息（名称、日期、优先级、耗时、标签等）
- 一键删除所有已完成任务

### 2. 视图切换
- **时间轴视图**：按日期分组显示任务卡片，支持**按相似度智能分组**（基于中文 bigram Jaccard 相似度）
- **流程图视图**：展示任务间的依赖关系（基于 Vue Flow）

### 3. 决策树（任务内执行方案）
- 每个任务卡片均可打开**全屏决策树流程图**弹窗
- 使用分层布局算法自动生成节点位置
- 支持节点增删改、勾选完成状态、分支条件标注
- 通过 **AI 提示词生成** 功能从自然语言描述自动创建决策树 JSON
- 画布背景为柔和的灰色网格（rgba(180,180,180,0.15)），便于视觉定位

### 4. 数据持久化
- 所有任务及决策树数据保存在 `localStorage`
- 支持 JSON 文件导入/导出

## 技术栈
- Vue 3 + `<script setup>`
- Element Plus UI 组件
- Vue Flow（流程图渲染）
- 自定义分层布局算法
