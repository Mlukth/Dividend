# 三角色辩论馆

一个基于 Vue 3 的轻量级多角色辩论辅助工具，通过模拟攻/防/中立三方对话，帮助用户梳理复杂观点，并借助 DeepSeek AI 填补发言、润色表达、生成观点结构地图。

## 项目简介
- **用途**：以“辩论赛”形式组织自己的思路，手动扮演 A（攻方）、B（防方）、C（观察员）三个角色进行交替发言，同时可调用 AI 自动生成反驳内容或润色已有发言。
- **核心特色**：支持一键将当前辩论历史转化为 **Mermaid 观点树（地图）**，可缩放、平移、全屏查看，直观展示论点攻防关系和潜在讨论分支。
- **数据持久化**：支持导出为 JSON 文件，方便存档、分享或继续编辑。

## 技术栈
- 前端框架：Vue 3（Composition API）
- UI：单文件组件（Scoped CSS），无额外 UI 库
- 依赖：`uuid`（生成消息ID）、`mermaid`（渲染观点树）
- API：DeepSeek Chat API

## 安装与运行
1. 确保已安装 Node.js（≥16）
2. 将 `三角色辩论馆.vue` 放置于 Vue 3 项目（如 Vite 搭建）的 `src/components/` 目录下，并在 `App.vue` 中引入：
   ```vue
   <template>
     <DebateApp />
   </template>
   <script setup>
   import DebateApp from './components/三角色辩论馆.vue'
   </script>
   ```
3. 安装依赖：
   ```bash
   npm install uuid mermaid
   ```
4. 启动项目，在界面底部的输入框中填入 DeepSeek API Key（自动保存至 localStorage）。

## 界面与操作说明
### 顶部工具栏
- **角色选择**：点击“角色A/角色B/角色C”按钮切换当前发言身份。
- **观点树生成**：点击“🌳 生成观点树”基于现有对话调用 AI 生成 Mermaid 脑图（左侧面板展示）。
- **导入/导出**：将当前对话及设置保存为 JSON 文件，或从 JSON 文件恢复对话。

### 左侧面板（观点地图）
- 显示 Mermaid 渲染的观点树（需先生成）。
- 支持 **Ctrl+滚轮** 缩放、**左键拖拽** 平移。
- 右上角按钮可调整缩放比例，点击“🔍”进入全屏模式（浅灰背景，同样支持缩放和拖动）。

### 右侧区域（对话流）
- 显示所有按时间顺序排列的发言，不同角色有不同左侧色条。
- 每条消息可 **✏️编辑**、**🗑删除**。
- 对于手动输入的发言，提供 **✨润色** 按钮，AI 会返回带修订标记的润色结果（红色删除线+绿色下划线）。
- 输入框可填写当前角色发言，按 `Ctrl+Enter` 发送。

### AI 辅助功能
- **AI 填补**：点击“🤖 AI填补”让 AI 根据对话历史以当前角色身份生成一句反驳发言（消息带绿色 AI 标签）。
- **AI 润色**：对某条手动发言执行改写，保持原意但更严谨流畅，并显示对比修订。

## 提示词配置
点击“📝 提示词”可展开编辑器，共三套提示模板：
- **填补发言**：用于生成反驳内容，支持变量 `{{topic}}`、`{{role}}`、`{{stance}}`、`{{history}}`
- **润色修改**：用于改写发言，变量 `{{content}}`
- **观点树生成**：要求 AI 输出 Mermaid mindmap 格式，变量 `{{history}}`

提示词会自动保存至项目 JSON 导出文件中，导入时可还原。

## 主要函数说明（供后续迭代参考）
| 函数名 | 作用 |
|--------|------|
| `makeMsg()` | 创建消息对象，包含 id、role、source、修订标记等字段 |
| `renderContent()` | 将消息内容渲染为 HTML，解析修订标记并高亮 |
| `send()` | 手动发送一条消息 |
| `editMsg()` / `deleteMsg()` | 编辑或删除指定消息 |
| `callDeepSeek()` | 封装 DeepSeek API 请求，返回模型回复 |
| `fillMessage()` | 调用 AI 填补发言，追加到消息列表 |
| `polishMessage()` | 调用 AI 润色发言，更新消息的修订标记和内容 |
| `generateTree()` | 组装对话历史，请求 AI 生成 Mermaid 代码，并渲染到左侧面板 |
| `exportJSON()` / `importJSON()` | 导出/导入整个项目的对话、提示词和观点树代码 |
| `zoomIn/Out/resetZoom()` | 控制观点树缩放比例 |
| `startPan/onPan/endPan()` | 处理左键/右键拖动平移 |
| `renderMermaidTo()` | 将 Mermaid 代码渲染至指定容器 |
| `openFullscreen/closeFullscreen()` | 全屏查看观点树 |

## 注意事项
- 使用前必须填写有效的 DeepSeek API Key，否则 AI 功能无法使用。
- 观点树依赖 AI 返回合规的 Mermaid mindmap 代码，若解析失败可检查提示词模板或手动调整返回内容。
- 项目数据保存在浏览器 localStorage 中，更换设备需通过导入/导出迁移。
